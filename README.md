# TheOtServer Client

The official game client for **[TheOtServer](https://theotserver.com)** — a custom-skinned build of **[OTClientV8](https://github.com/OTCv8/otclientv8)** for Open Tibia (Tibia 8.60 protocol).

## Download & play

Grab the latest version from the **[Releases page](https://github.com/Mizodawalaeh/theotserver-client/releases/latest)**, unzip it, and run **`otclient_gl.exe`** (or `otclient_dx.exe` if you hit graphics issues). It connects to `play.theotserver.com:7171` out of the box — create an account at [theotserver.com](https://theotserver.com) and log in.

## What's customized

A rebranded front door: login screen, character list, dialog windows, the MedievalSharp display font, and TheOtServer icons. The underlying engine is stock OTClientV8.
